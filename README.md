# 🧮 Calculadora en Java (Swing)

Aplicación de escritorio desarrollada en **Java** utilizando **Swing** para la interfaz gráfica.  
La calculadora permite realizar operaciones matemáticas básicas mediante botones interactivos.

## 📌 Características

- Interfaz gráfica construida con **Java Swing**
- Operaciones básicas:
  - Suma (+)
  - Resta (-)
  - Multiplicación (*)
  - División (/)
- Botón para limpiar la pantalla (`C`)
- Manejo de eventos con `ActionListener`
- Interfaz organizada con `BorderLayout` y `GridLayout`
- Uso de **Look and Feel (JTattoo)** para mejorar la apariencia visual

## 🛠️ Tecnologías utilizadas

- Java
- Swing
- AWT
- JTattoo Look and Feel

## 📂 Estructura del proyecto
src/
calculadora/
Calculadora.java

El proyecto está implementado en **una sola clase Java**, que contiene:

- Creación de la interfaz gráfica
- Creación dinámica de botones
- Manejo de eventos de usuario
- Lógica de operaciones matemáticas
